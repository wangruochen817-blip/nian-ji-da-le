# /other-2 Workflow Agent Design

**目标**

新增一个独立页面 `/other-2`，按参考图 `screenshot-20260625-220539.png` 复刻框图结构、文案和对齐关系，同时保持当前站点的深色视觉基调。现有 `/other` 页面不可被覆盖或改形。

**范围**

- 新增页面组件 `src/pages/Other2.tsx`
- 在 `src/App.tsx` 增加 `/other-2` 路由
- 在 `src/App.test.tsx` 增加路由级回归测试

**设计约束**

- 左侧主图必须保留四个纵向标签区：`场景`、`技能`、`工具`、`原子能力`
- 顶部三条场景条、中央三块技能面板、底部两列原子能力面板，按参考图的层级和对齐关系实现
- 右侧为独立纵向说明栏，标题为 `Workflow Agent`，下方四张粉色卡片，卡片高度比例与参考图一致
- 文字内容按参考图填写
- 允许在字体、阴影、底色透明度上向当前站点风格靠拢，但不得改变框架关系

**实现方式**

延续现有 `Other.tsx` 的做法，使用纯 React + Tailwind 进行手工布局，不引入新的状态逻辑或外部依赖。为保持代码边界清晰，此页面作为独立页面文件实现，只复用现有通用样式组件，不回写或重构 `/other` 页面。

**验证**

- 路由测试验证 `/other-2` 能渲染关键标题与文案
- `npm test` 验证回归
- `npm run build` 验证页面可构建
